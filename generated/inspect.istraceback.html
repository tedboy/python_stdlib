1.7.1.45. inspect.istraceback¶

inspect.istraceback(object)[source]¶

Return true if the object is a traceback.

Traceback objects provide these attributes:
tb_frame frame object at this level tb_lasti index of last attempted instruction in bytecode tb_lineno current line number in Python source code tb_next next inner traceback object (called by this level)
