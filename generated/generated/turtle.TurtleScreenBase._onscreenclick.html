turtle.TurtleScreenBase._onscreenclick¶

TurtleScreenBase._onscreenclick(fun, num=1, add=None)[source]¶

Bind fun to mouse-click event on canvas. fun must be a function with two arguments, the coordinates of the clicked point on the canvas. num, the number of the mouse-button defaults to 1

If a turtle is clicked, first _onclick-event will be performed, then _onscreensclick-event.
