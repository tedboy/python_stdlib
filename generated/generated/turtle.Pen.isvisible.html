turtle.Pen.isvisible¶

Pen.isvisible()¶

Return True if the Turtle is shown, False if it’s hidden.

No argument.

Example (for a Turtle instance named turtle): >>> turtle.hideturtle() >>> print turtle.isvisible(): False
