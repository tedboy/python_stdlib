smtplib.SMTP.__init__¶

SMTP.__init__(host='', port=0, local_hostname=None, timeout=<object object>)[source]¶

Initialize a new instance.

If specified, host’ is the name of the remote host to which to connect. If specified, `port’ specifies the port to which to connect. By default, smtplib.SMTP_PORT is used. If a host is specified the connect method is called, and if it returns anything other than a success code an SMTPConnectError is raised. If specified, `local_hostname is used as the FQDN of the local host for the HELO/EHLO command. Otherwise, the local hostname is found using socket.getfqdn().
