poplib.POP3_SSL¶

class poplib.POP3_SSL(host, port=995, keyfile=None, certfile=None)[source]¶

POP3 client class over SSL connection

Instantiate with: POP3_SSL(hostname, port=995, keyfile=None, certfile=None)

hostname - the hostname of the pop3 over ssl server port - port number keyfile - PEM formatted file that contains your private key certfile - PEM formatted certificate chain file

See the methods of the parent class POP3 for more documentation.
