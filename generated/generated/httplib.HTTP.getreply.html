httplib.HTTP.getreply¶

HTTP.getreply(buffering=False)[source]¶

Compat definition since superclass does not define it.

Returns a tuple consisting of: - server status code (e.g. ‘200’ if all goes well) - server “reason” corresponding to status code - any RFC822 headers in the response from the server
