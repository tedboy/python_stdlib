turtle.dot()¶

turtle.dot(size=None, *color)¶

Draw a dot with diameter size, using color.

Optional arguments: size – an integer >= 1 (if given) color – a colorstring or a numeric color tuple

Draw a circular dot with diameter size, using color. If size is not given, the maximum of pensize+4 and 2*pensize is used.

Example: >>> dot() >>> fd(50); dot(20, “blue”); fd(50)
