linecache¶

Cache lines from files.

This is intended to read lines from modules imported – hence if a filename is not found, it will look down the module search path for a file by that name.

Functions¶

checkcache([filename]) Discard cache entries that are out of date.
clearcache() Clear the cache entirely.
getline(filename, lineno[, module_globals])
getlines(filename[, module_globals]) Get the lines for a file from the cache.
updatecache(filename[, module_globals]) Update a cache entry and return its list of lines.
