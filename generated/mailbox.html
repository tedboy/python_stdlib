mailbox¶

Read/write support for Maildir, mbox, MH, Babyl, and MMDF mailboxes.

MMDFMessage([message]) Message with MMDF-specific properties.

Exceptions¶

Error Raised for module-specific errors.
ExternalClashError Another process caused an action to fail.
FormatError A file appears to have an invalid format.
NoSuchMailboxError The specified mailbox does not exist and won’t be created.
NotEmptyError The specified mailbox is not empty and deletion was requested.
