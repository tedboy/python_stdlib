dis¶

Disassembler of Python byte code into mnemonics.

Functions¶

dis([x]) Disassemble classes, methods, functions, or code.
disassemble(co[, lasti]) Disassemble a code object.
disassemble_string(code[, lasti, varnames, ...])
disco(co[, lasti]) Disassemble a code object.
distb([tb]) Disassemble a traceback (default: last traceback).
findlabels(code) Detect all offsets in a byte code which are jump targets.
findlinestarts(code) Find the offsets in a byte code which are start of lines in the source.
